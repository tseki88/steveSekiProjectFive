# Boardgame Tool Kit

## Live Link
http://tseki88.github.io/steveSekiProjectFive

### Description
Board Game Tool Kit: 
An all in one app with multiple components that can be added based on the board game type.

### Currently Available Tools
 - Count Down Timer
 - Stop Watch
 - Dice Rolling [4,6,8,10,12,20]
